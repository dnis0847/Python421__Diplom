{{ course.description }}

Чему вы научитесь:

  • Основам React и его экосистемы
  • Созданию компонентов и управлению их жизненным циклом
  • Работе с хуками и функциональными компонентами
  • Управлению состоянием приложения с помощью Redux
  • Маршрутизации в React-приложениях
  • Оптимизации производительности React-приложений

Предварительные требования:

Базовые знания HTML, CSS и JavaScript. Опыт работы с современными инструментами разработки (npm, Git) будет полезен, но не обязателен.

Программа курса:

  1. Введение в React и его экосистему
  2. Компоненты и их жизненный цикл
  3. Работа с пропсами и состоянием
  4. Хуки в React
  5. Маршрутизация с React Router
  6. Управление состоянием с Redux
  7. Работа с формами в React
  8. Оптимизация производительности
  9. Тестирование React-приложений
  10. Развертывание React-приложений
